9.3 / FAN-OUT
02 · El fundamento

¿Por qué necesitamos un evento completo?

Evitando el re-acoplamiento

❌ Evento incompleto

Solo enviamos el ID de la orden

{ "orderNumber": "ORD-123" }

Problema: Inventory tiene que llamar de vuelta a Order-Service. Esto es lento y frágil.

✓ Evento completo

Payload autosuficiente

{ "orderNumber": "ORD-123", "email": "user@email.com", "items": [ {"sku": "iphone", "qty": 2} ] }

Solución: Inventory tiene todo lo que necesita para trabajar solo.

9.3 / FUNDAMENTO
03 · Diseño del Contrato

¿Por qué usar Nested Records?

Estructura limpia y portable

Principio 1

Alta Cohesión

El "Item de Evento" no existe por sí solo. Depende totalmente del evento padre. Anidarlo refleja esta relación fuerte.

Principio 2

Atomicidad

Al tener todo en un solo archivo, garantizamos que la definición del esquema sea indivisible. Quien tenga el evento, tiene la estructura completa.

Creando el evento en Order-Service

Paquete: com.ecommerce.orderservice.event

package com.ecommerce.orderservice.event;

import java.util.List;

// Evento Principal: Representa el mensaje JSON raíz
public record OrderPlacedEvent(
    String orderNumber,
    String email,
    List<OrderItemEvent> items
) {

    // Record Anidado (Static Nested): Define la estructura
    // de los objetos dentro del array.
    public record OrderItemEvent(
        String sku,
        String price,
        Integer quantity
    ) {
    }
}
9.3 / CÓDIGO
05 · Pruebas

Método main para pruebas

Validando la estructura in-situ

// --- MÉTODO MAIN: SOLO PARA APRENDIZAJE ---
public static void main(String[] args) {
    // 1. Crear los items (accediendo al record estático interno)
    var item1 = new OrderPlacedEvent.OrderItemEvent(
        "iphone-15", "1200.00", 1
    );

    // 2. Crear el evento completo
    var evento = new OrderPlacedEvent(
        "ORD-123",
        "cliente@email.com",
        List.of(item1)
    );

    // 3. Simular la lectura
    System.out.println("--- PROCESANDO EVENTO ---");
    for (OrderItemEvent item : evento.items()) {
        System.out.println("Descontando Stock -> SKU: " +
            item.sku());
    }
}
